The topic I looked into is:
What are the best practices associated with using classes vs. ids?

First of all, Classes are reusable attributes that can be added to HTML elements to bring different styles.
ID attributes work in the same manner as classes do. However, you have to keep in mind that with ID attributes you can only use it to style a single element only.
So if you have one element that you wish to have specific styling then use ID attribute.
